## Amlogic Service

Install to EMMC: </br>
Login to `OpenWrt` → `System` → `Amlogic Service` → `Install OpenWrt`

Online Update: </br>
Login to `OpenWrt` → `System` → `Amlogic Service` → `Online Download Update`

Manual Update: </br>
Login to `OpenWrt` → `System` → `Amlogic Service` → `Manually Upload Update`
